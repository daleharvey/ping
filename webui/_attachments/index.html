Frequently Asked Questions

How do I install?

All you need to do is embed this image tag anywhere into your HTML. 

<img src="//xxxxxxx.com/spacer.gif" />

Is this Open Source? Show me the code

Ok: http://github.com/daleharvey/ping

I already have Google Analytics?

Google Analytics only tracks your own website, that is certainly important and taking your audience into account should be the most important factor when deciding what technologies to use, however its also useful to have information about the trends happening in global browser usage.

Isnt this the same as StatCounter?

Stat counter is a full analytics package which is expensive in terms of page load, we hope by providing an ultra lightweight mechanism to contribute, people who are using other more popular analytics tools such as Google Analytics will also take part.

What about Javascript statisics / Monitor resolution and other statistics?

In order to make it as lightweight as possible, we only collect statistics which can be pulled from a plain image request, that makes sure we have as little impact on your page as possible.

But my privacy!

We only collect very minimal amount of data from the requests, and your site details are never recorded at all.

Got API?

All data is stored in CouchDB and is accessible though the CouchDB API, you can browse the data in futon.

Can I run this on a personal site?

You probably dont want to, there are better free analytics tools like Google Analytics and StatCounter. However if you really want to please just checkout the source from Github and follow the README, if you are awesome you can replicate your results into the global database.

Is this going to slow down my site?

We only use a single pixel tracking image so there should be no noticeable effect on your website even if our server doesnt respond, however your visitors will be making another HTTP request so it isnt completely without effect.
